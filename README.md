# OS-23Fall-FDU

Course Website for OS Autumn 2023 at Fudan University

这是复旦大学 2023 年秋季学期《操作系统》课程的配套实验内容。我们将建立一个基于 ARM 架构的简易教学操作系统。

暂定的实验内容将包括：

* 引导 Booting
* 内存管理
* 进程管理
* 多核管理
* 中断与异常、缺页处理
* 块设备驱动
* 文件系统
* IPC
* Shell
